기능 요구사항
- 로그인(자동로그인), 회원가입, 로그아웃, 회원탈퇴, 비밀번호 찾기(질문-답변)
- 회원 정보 수정(닉네임, 비밀번호 수정)
- 팀장 : IP주소 입력(자동 저장),그룹 생성,그룹 초대, 그룹 삭제, 강퇴 기능, 연결 기능, 연결 종료,제어 권한 가져오기
- 팀원 : 그룹 참가 요청, 연결 기능, 제어 권한 가져오기, 연결 종료
- 그룹 1개 최대 20명

UI 프로토타입
- Splash Activity
- SignIn Activity (로그인 화면, 회원가입 버튼)
- SignUp Activity (회원가입 화면)
- Main Activity (그룹 관리, 프로필 보기, 설정)
- GroupManagement Fragment (그룹 생성, 그룹 삭제)
- Profile Fragment (비밀번호 수정, 닉네임 수정, 회원탈퇴)
- Setting Fragment (키보드 커스텀, 마우스 커스텀-휠감도, 마우스감도)
- KeyboardCustom Activity
- Group Activity (초대, 강퇴, 연결 버튼, ip입력)
- Control Activity (MouseFragment, Keyboard Fragment)
